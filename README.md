# task_api

FastAPI backend for a simple Task / Notes mobile application.

## Features
- Task CRUD APIs
- Clean layered architecture (routes, schemas, services, models)
- SQLite + SQLAlchemy
- Pydantic v1
- OpenAPI / Swagger documentation
- Health check endpoint

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- Pydantic v1
- Uvicorn
